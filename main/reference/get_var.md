# Get object from `qenv`

**\[deprecated\]** Instead of `get_var()` use native R
operators/functions: `x[[name]]`, `x$name` or
[`get()`](https://rdrr.io/r/base/get.html):

## Usage

``` r
get_var(...)

# S3 method for class 'qenv.error'
x[[i]]
```

## Arguments

- ...:

  function is deprecated.

- x:

  (`qenv`)

- i:

  (`character(1)`) variable name.
