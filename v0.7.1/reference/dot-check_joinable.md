# If two `qenv` can be joined

Checks if two `qenv` objects can be combined. For more information,
please see
[`join`](https://insightsengineering.github.io/teal.code/reference/join.md)

## Usage

``` r
.check_joinable(x, y)
```

## Arguments

- x:

  (`qenv`)

- y:

  (`qenv`)

## Value

`TRUE` if able to join or `character` used to print error message.
